## 导入模块

```python
01: 导入模块
import 模块1
import 模块2

02： 使用as
指代模块名称
import 模块1 as 模块别名

模块别民的取名方法： 大驼峰命名方法。

03： 导入模块

from ... import ...

04: 导入模块

from ... import *


python 每一个模块都有一个内置属性 __file__ 可以查询模块的完整路径


05： __name__ 内置属性 记录着一个字符串
如果被其他文件导入，__name__ 就是一个模块名
如果是当前执行的程序 __name__ 是__main__

```
